This application uses ES6 modules which cannot be loaded directly from the file system due to browser security restrictions.

Solutions:

  1. Use VS Code Live Server: Install the "Live Server" extension in VS Code, then right-click index.html and select "Open with Live Server"
  2. Use Python HTTP Server: Open terminal in this folder and run:
    python -m http.server 8000
    Then visit http://localhost:8000
  3. Use Node.js HTTP Server: Install globally with npm install -g http-server
    Then run http-server in this folder

This message will not appear when running through a web server.
